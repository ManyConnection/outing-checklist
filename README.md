# おでかけチェック 🎒

外出前の持ち物を場面別にチェック。通勤・旅行・ジムなど、シーン別リストで忘れ物ゼロを実現します。

## 機能

1. **シーン別チェックリスト** - 通勤、旅行、ジム、デートなど、シーンに合わせたプリセットリスト
2. **オリジナルリスト作成** - 自分だけのカスタムチェックリストを作成
3. **リマインダー通知** - 外出時間に合わせた通知でチェック忘れを防止
4. **統計表示** - よく忘れるアイテムや達成率を可視化

## 画面構成

- **ホーム** - シーン別リスト一覧とクイックチェック
- **チェック** - スワイプでアイテムをチェック
- **リスト編集** - アイテムの追加・削除・並べ替え
- **統計** - 忘れ物傾向と達成率
- **設定** - リマインダー・テンプレート管理

## 技術スタック

- React Native (Expo)
- TypeScript
- expo-router (ファイルベースルーティング)
- AsyncStorage (ローカルデータ保存)
- expo-notifications (プッシュ通知)
- expo-haptics (触覚フィードバック)

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npx expo start
```

## ビルド

```bash
# iOS
npx expo run:ios

# Android
npx expo run:android
```
